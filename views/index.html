Timestamp Microservice

Input any date to get the number of milliseconds elapsed since January 1, 1970, 00:00:00 UTC

Or leave empty to submit current date

Example Output:

{"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}

API Endpoint
